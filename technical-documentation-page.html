Introduction

Python is a versatile and popular computer programming language. Python can be used in web development, web applications, even to create graphics and games.

Like other programming languages, python has variables, comments, conditional loops, functions, classes including many in-built functions useful for creating programs aimed at a particular tasks.

Below is a breif look on Python.

Python Example

As is tradition, lets look at the "Hello World" program written in Python -

print("Hello World!")

That's it! As simple as that. Try running the code in the console and see the output.

Python Comments

Comments in Python by starting with "#". Start the line with it and Python will ignore the characters that follow. Below is an example.

#This is the comment print("Hello World!") #This is Code

Try running the code in the console and see the output.

Python Variables

Variables in Python are created when they are assigned a value.

x = 5 y = "Thor" print(x) print(y)

Variable type need not be specified when declaring and they can also change types after the value assignment.

x = 10 y = "Mighty Thor" print(x) print(y)
Python Data Types

Python has the following data types that are built-in.

  • String - x = "code"
  • Integer - x = 10
  • Float - x = 3.14
  • Boolean - x = True; y = False;
  • None - x = None
